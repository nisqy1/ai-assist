# 🤖 Projeto Assistente Virtual (AI Assistant)

Projeto de um Assistente Virtual desenvolvido em Python com foco em automação, reconhecimento de voz e interação por comandos de áudio.

Este projeto foi desenvolvido como desafio da DIO (Digital Innovation One), explorando conceitos de Inteligência Artificial e automação de tarefas.

---

# 📌 Funcionalidades

- 🎤 Reconhecimento de voz
- 🔊 Respostas por áudio (Text-to-Speech)
- 🌐 Pesquisa automática na web
- 📂 Abertura de aplicativos e arquivos
- ⏰ Informações de data e hora
- 💬 Interação com o usuário via comandos de voz
- 🧠 Automação básica utilizando IA

---

# 🚀 Tecnologias Utilizadas

## Linguagem

- SpeechRecognition

---

# 🛠️ Como Funciona

O assistente utiliza o microfone para capturar comandos de voz do usuário, converte o áudio em texto e executa ações previamente programadas.

Exemplos de ações:

- Abrir sites
- Realizar pesquisas no Google
- Informar horário atual
- Executar aplicativos locais
- Responder com áudio
